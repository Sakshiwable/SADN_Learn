# 🎓 SADN Learn

A full-stack learning platform that supports email verification and user role-based navigation (Admin & Learner).

## 🛠️ Technologies Used

### Client (Frontend)
- React.js
- Tailwind CSS
- Axios
- React Router DOM
- Toastify

### Server (Backend)
- Node.js
- Express.js
- MongoDB with Mongoose
- JWT (JSON Web Token)
- bcryptjs
- Nodemailer
- dotenv

---

## ✨ Features

- 🔐 **Signup/Login with Email Verification**
- 🧾 **JWT Authentication**
- 👩‍🏫 **Role-Based Access (Admin, Learner)**
- 📬 **Email sent via Gmail SMTP**
- 📊 **Progress & Section Tracking**
- 💻 **React Frontend + Express API Backend**

---

## 🚀 How to Run Locally

### Prerequisites:
- Node.js and npm
- MongoDB (local or Atlas)
- Gmail account (with App Password enabled)
